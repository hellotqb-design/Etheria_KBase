**【机制讲解】**

兵祸的机制非常简单粗暴，首先是全难度通用的【击杀再动+回合叠攻】，前者为实打实的萌新杀手，队伍只要减员，就会被兵祸一套Aoe组合拳直接送走，后者则是“输出检测”，数值不够被拉入长线作战，就会被兵祸缓慢上涨的水温活活烫死，想轻松攻略，就得在这两大机制上做对策。

先来看【击杀再动】，“炼狱”难度之前，兵祸的进攻手只会上“防御下降”debuff，携带任意奶妈就可以很好的保证队伍生存，直接废掉兵祸的再动，可选的奶妈有“缬草（平均血量、高额护盾、短暂无敌）”“善雅（持续回复、生命提升）”“千明（持续回复、高额减伤）”，除了携带奶妈，也可以让输出优先选中进攻手，击杀之后能大大减少Boss输出，方便后续击杀本体

| 击杀再动 | 携带奶妈，保证队伍生存 | 可选角色： | | 缬草、 | | 善雅、 | | 千明（非炼狱） |
| 其他可选的生存： | | 维罗妮卡、 | | 米娅、 | | 拂秋、 | | 赫尔基德（不推荐） |
| Tips：赫尔基德3技能虽然能提供护盾，但整体数值和其攻击力有关，想要在PVE使用，核心刚需2个攻击，在这种养成条件下，护盾依旧不尽人意，不推荐使用 |

再来看【回合叠攻】，常规思路是“快速通关”，不让重构兵祸叠攻击力，可选的方案有【高伤速杀】【加速/推拉条】，要么直接跟boss爆了，要么通过技能效果，制造更多我方回合，减少boss出手次数，box较差，缺少上述角色的情况下，可以选择邪门思路，通过嘲讽角色，强制兵祸释放1技能，减少释放加攻2技能的次数，这里推荐使用R卡恶命，技能加点便宜的同时，还能兼顾回复、推条，后续在虚烬探索也能高频就业，超级保值。

解决上述两种机制，通关地狱难度不成问题，但到了炼狱难度，全都不一样了，Boss开局就有3回合的【免疫】，buff存在时推条、控制均不生效，免疫掉了也有60%的“减少行动进度”效果抵抗，没法通过推条无脑击杀boss。攻击手debuff追加【禁疗】，没有净化根本抬不动血线，再加上克属增伤，对【异】属性角色造成额外200%伤害，直接废掉最稳定好用的净化奶妈凌罗，所以，需要根据新增机制，重新组一套适配炼狱的队伍。

开局免疫的解决方法非常简单，选用1动驱散角色即可，游戏内大部分角色（除梨梨之外）都会优先释放3技能，点进角色界面，看到3技能下标有驱散的非【异】属性角色即可编入队伍，没有SSR的情况下，可以选择SR品质角色“水月真”

| 开局免疫 | 驱散清除，方便后续输出 | 可选角色： | | 水月真 |
| 更好用的驱散： | | 海因里希、 | | 维罗妮卡、 | | 赫同 |

禁疗buff就比较难解决了，除了禁疗，兵祸还会施加防御、速度下降等debuff，常规净化角色没法准确解掉禁疗，就会被兵祸持续压低血线，直到完成斩杀，释放Aoe组合拳。比较笨的解决方法可以用善雅、米娅这种针对debuff的奶妈硬着头皮解决，也可以在队伍中编入能施加“免疫”的角色，减少其他debuff的数量，让前面的奶妈能精准净化禁疗，上述都是直伤长线队伍的解法，像Dot这种偏短线的队伍，也可以直接选用护盾角色，硬吃boss伤害

**【发育流程】**

看完了boss机制，让我们来稍微梳理一下发育流程

普通~危机难度，对角色速度没有太高要求，完全可以靠赠送源器、紫色智壳的数值硬莽过去，过了之后，拿首通奖励和进度奖励赠送的源器去刷“源器-多琪”副本，挑3词条以上带速度的“疾驰”装备强化，尽可能拉高角色速度，方便后续噩梦难度的击杀

噩梦重构兵祸的门槛不高，有一个满级黄昏骑士就能轻松击杀，从地狱难度开始，副本对源器、智壳数值的要求就更高了，此时可以去地狱难度的“智壳-尼可娜娜”，刷两个满级的匠来增加队伍生存，刷一个满级白撩武者提高队伍输出，还是打不动boss，可以适当洗一下智壳副词条，或者去“源器-奥罗拉”刷一点“狂怒”装备，进一步提高队伍输出，也可以直接转投Dot队，拉满队伍生存，用真伤逃课。

在挑战炼狱副本之前，最好在智壳、源器本多沉淀一段时间，等速度最快的角色超过261之后，再尝试通关，初见推荐使用Dot队，输出压力小，可以大胆做生存、命中
