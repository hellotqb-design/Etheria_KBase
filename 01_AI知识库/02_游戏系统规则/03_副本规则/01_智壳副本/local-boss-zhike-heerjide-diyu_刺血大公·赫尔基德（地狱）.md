## 技能信息

## 技能一：恶堕黑赫

### 原始技能信息

- 技能名称：恶堕黑赫
- 技能标签：群伤 / 减益
- 技能效果：对敌方全体造成500%伤害，100%概率为目标附加2回合【防御下降】。

### 关联状态：防御下降

- 状态名称：防御下降
- 状态效果：防御下降60%。

### 机制解读

该技能是高倍率群体伤害加防御削弱。被附加【防御下降】后，队伍承受后续群体伤害的压力会明显提高。地狱难度下建议准备净化、免疫、护盾、治疗、减伤或高生存配置。

---

## 技能二：巨型帷幕

### 原始技能信息

- 技能名称：巨型帷幕
- 技能标签：群伤 / 减益
- 技能效果：对敌方全体造成600%伤害，100%概率为目标附加2回合【速度下降】，并为自身附加5层的【巨型帷幕】。

### 关联状态：速度下降

- 状态名称：速度下降
- 状态效果：速度降低30%。

### 关联状态：巨型帷幕

- 状态名称：巨型帷幕
- 状态效果：持续期间，攻击自身的首段伤害减免50%。最多附加10层，不可被驱散，每受到1次技能伤害减少1层。

### 机制解读

【巨型帷幕】是地狱难度下的核心机制之一。它会让攻击赫尔基德的首段伤害减免50%，最多10层，且不可驱散。每受到1次技能伤害减少1层，因此多段技能或高频技能伤害更适合消耗帷幕层数。单段爆发在帷幕存在时会受到明显影响。

---

## 技能三：荆棘

### 原始技能信息

- 技能名称：荆棘
- 技能标签：被动
- 技能效果：赫尔基德攻击时有50%概率击退目标15%行动进度，并且自身每有1层【巨型帷幕】，伤害提升10%。

### 关联状态：巨型帷幕

- 状态名称：巨型帷幕
- 状态效果：持续期间，攻击自身的首段伤害减免50%。最多附加10层，不可被驱散，每受到1次技能伤害减少1层。

### 机制解读

该被动让赫尔基德具备行动条干扰能力，同时【巨型帷幕】层数越高，赫尔基德伤害越高。若赫尔基德身上有5层【巨型帷幕】，则伤害提升50%；若叠至10层，则伤害提升100%。因此队伍需要尽快通过技能伤害消耗帷幕层数，避免 Boss 同时获得减伤和增伤。

## 应对思路

### 推荐队伍方向

| 推荐方向 | 原因 |
|---|---|
| 高生存队伍 | Boss 具备500%和600%的群体伤害技能 |
| 净化 / 免疫角色 | 用于处理【防御下降】和【速度下降】 |
| 护盾 / 减伤 / 治疗 | 用于承受连续群体伤害 |
| 多段技能输出 | 可以更快消耗【巨型帷幕】层数 |
| 高频技能输出 | 【巨型帷幕】每受到1次技能伤害减少1层 |
| 稳定直伤输出 | 帷幕不可驱散，需要持续通过技能伤害压层 |
| 有速度循环能力的角色 | 可降低【速度下降】和击退行动进度带来的节奏损失 |

### 不推荐队伍方向

| 不推荐方向 | 原因 |
|---|---|
| 单段大核爆 | 容易被【巨型帷幕】首段50%减伤影响 |
| 缺少净化或免疫的脆皮队 | 被【防御下降】后容易被后续群伤击穿 |
| 完全依赖速度顺序的队伍 | Boss 会附加【速度下降】，并有概率击退行动进度 |
| 依赖驱散 Boss 增益的打法 | 【巨型帷幕】不可被驱散 |

## 配装建议

### 输出位配装方向

优先考虑：

- 多段技能输出
- 高频技能输出
- 稳定直伤输出
- 能在【巨型帷幕】存在时继续消耗层数
- 不过度依赖单段首击爆发

推荐属性倾向：

- 输出主属性按角色自身倍率决定
- 暴击率 / 暴击伤害适合直伤角色
- 速度可提高技能循环和破帷幕频率
- 若角色本身有多段技能或追击机制，可优先作为破层候选

### 辅助位配装方向

优先考虑：

- 速度
- 生存
- 效果抵抗
- 治疗量
- 护盾量
- 净化频率
- 免疫覆盖率

辅助位主要任务：

1. 保证队伍不被【防御下降】后连续击杀。
2. 处理【速度下降】导致的行动节奏崩坏。
3. 帮助输出位尽快消耗【巨型帷幕】。
4. 维持队伍在高压群伤下的存活。

### 生存位配装方向

优先考虑：

- 生命
- 防御
- 减伤
- 护盾
- 治疗
- 效果抵抗

注意事项：

- Boss 会附加【防御下降】，纯堆防御可能不稳定。
- 队伍最好具备净化或免疫，否则防御下降后容易被连续群伤击穿。
- 如果角色依赖速度循环，需要注意【速度下降】会降低行动频率。

## AI 调号器判断规则

当用户询问“这个 Boss 怎么打”“这个角色能不能打赫尔基德地狱”“我的配装怎么调”时，AI 应按以下顺序判断：

1. 判断队伍是否能承受500%和600%群体伤害。
2. 判断队伍是否有净化或免疫，用于处理【防御下降】和【速度下降】。
3. 判断输出是否过度依赖单段爆发；若是，需要提醒会被【巨型帷幕】首段减伤影响。
4. 判断队伍是否有多段技能或高频技能伤害；若缺少，处理【巨型帷幕】会较慢。
5. 判断队伍是否依赖严格配速；若依赖，需要提醒 Boss 会附加【速度下降】并击退行动进度。
6. 判断是否试图驱散【巨型帷幕】；若是，需要提醒【巨型帷幕】不可被驱散。
7. 如果用户询问控制、推条或持续伤害能否使用，需要说明：地狱截图未显示对应免疫规则，不能直接套用炼狱或修罗规则，应以实战或补充截图为准。
