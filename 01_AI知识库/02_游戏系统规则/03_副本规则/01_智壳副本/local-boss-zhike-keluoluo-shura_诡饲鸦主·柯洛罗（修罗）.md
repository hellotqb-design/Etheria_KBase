## 一句话总结

诡饲鸦主·柯洛罗是一个以防御倍率伤害、速度下降、石化、缚魂爆发为核心的高压 Boss。修罗难度下，柯洛罗受到【虚】相性异格者伤害增加100%，但柯洛罗和恐鸦免疫控制、减少行动进度和【持续伤害】，并会对【理】相性异格者造成额外200%伤害。

应对思路是：优先考虑【虚】相性直伤输出，避免使用【理】相性角色；不要依赖控制、推条或 Dot；队伍需要免疫、净化、速度循环、拉条和足够生存来处理【速度下降】、【石化】与【缚魂】到期引爆。

## 修罗难度特殊规则

### 原始规则文本

修罗难度下柯洛罗受到【虚】相性异格者伤害增加100%。

柯洛罗和恐鸦免疫所有控制效果、减少行动进度效果和【持续伤害】效果，并对【理】相性异格者造成额外200%伤害。

柯洛罗将有概率产出拥有3条矩阵的智壳。

为追求极致的力量，柯洛罗接受了原质融合实验，与「鬼面鸦」合为一体，化身为「诡饲鸦主」。如今，他以新的姿态卷土重来，誓要让曾经的敌手们付出代价。

## 技能信息

## 技能一：镰佑

### 原始技能信息

- 技能名称：镰佑
- 技能标签：单伤 / 减益
- 技能效果：对单体目标造成750%防御的伤害，并附加1回合【速度下降】。

### 关联状态：速度下降

- 状态名称：速度下降
- 状态效果：速度降低30%。

### 机制解读

该技能是高倍率防御伤害，并附带【速度下降】。它会压低单体角色行动频率，影响治疗、净化、拉条或输出循环。队伍需要免疫、净化或足够速度冗余来降低影响。

---

## 技能二：裂舍之影

### 原始技能信息

- 技能名称：裂舍之影
- 技能标签：群伤 / 控制 / 减益
- 技能效果：对全体造成3段250%防御的伤害，最后一击附加1回合【石化】。再次随机单体目标造成600%防御的伤害，并附加2回合【速度下降】。

### 关联状态：石化

- 状态名称：石化
- 状态效果：无法进行行动，且在自身回合结束时无法自动恢复冷却。

### 关联状态：速度下降

- 状态名称：速度下降
- 状态效果：速度降低30%。

### 机制解读

该技能是全队压力核心。它既有多段群体防御伤害，又会在最后一击附加【石化】，还会追加随机单体高倍率伤害和【速度下降】。

【石化】会让角色无法行动，并阻断回合结束时的自动冷却恢复，因此对依赖技能循环、净化、治疗、拉条的队伍威胁很高。建议提前覆盖免疫，或准备稳定净化与高生存。

---

## 技能三：幽镰追索

### 原始技能信息

- 技能名称：幽镰追索
- 技能标签：单伤 / 缚魂 / 自拉条
- 技能效果：对攻击最高的敌人造成300%防御的伤害，再为自身附加2回合【缚魂】并提升自身50%行动进度。

### 关联状态：缚魂

- 状态名称：缚魂
- 状态效果：到期移除时引爆，对全体敌人造成2000%防御的真实伤害；每次敌方异格者行动后，移除1层此效果。

### 机制解读

【缚魂】是本 Boss 的关键危险机制。若到期移除，会对全体造成高额真实伤害。由于每次敌方异格者行动后会移除1层，因此队伍需要提高自身行动频率，通过速度、拉条、自拉条或多角色快速行动来尽快消耗【缚魂】层数。

注意：这里推荐的是提高我方行动频率，不是减少 Boss 行动进度。Boss 在修罗难度下免疫减少行动进度效果，因此推条、偷条、击退 Boss 行动进度的打法不可作为核心应对。

---

## 技能四：诡饲鸦主

### 原始技能信息

- 技能名称：诡饲鸦主
- 技能标签：开场免疫
- 技能效果：开场时为自身附加3回合【免疫】。

### 关联状态：免疫

- 状态名称：免疫
- 状态效果：不会被附加任何减益效果，增加冷却或击退行动进度。

### 机制解读

该技能让柯洛罗开场拥有3回合【免疫】，开局阶段不要依赖减益、增加冷却或击退行动进度来建立优势。由于修罗规则还额外免疫控制、减少行动进度和【持续伤害】，控制队、推条队、Dot 队的有效性都很低。

## 应对思路

### 推荐队伍方向

| 推荐方向 | 原因 |
|---|---|
| 【虚】相性直伤输出 | 修罗难度下柯洛罗受到【虚】相性异格者伤害增加100% |
| 高生存队伍 | Boss 技能多为高倍率防御伤害，并有真实伤害爆发威胁 |
| 免疫 / 净化角色 | 用于处理【速度下降】和【石化】 |
| 高速 / 拉条 / 自拉条体系 | 通过提高我方行动频率来消耗【缚魂】层数 |
| 非 Dot 直伤输出 | Boss 免疫【持续伤害】效果 |
| 不依赖控制的输出体系 | Boss 免疫所有控制效果 |
| 不依赖推条的队伍 | Boss 免疫减少行动进度效果 |
| 能处理单点高压的治疗或护盾 | Boss 会随机单体追加600%防御伤害，并点名攻击最高目标 |

### 不推荐队伍方向

| 不推荐方向 | 原因 |
|---|---|
| 控制队 | 柯洛罗和恐鸦免疫所有控制效果 |
| 推条 / 偷条队 | 柯洛罗和恐鸦免疫减少行动进度效果 |
| Dot / 持续伤害队 | 柯洛罗和恐鸦免疫【持续伤害】效果 |
| 【理】相性角色 | 柯洛罗和恐鸦对【理】相性异格者造成额外200%伤害 |
| 开局减益爆发 | 柯洛罗开场自带3回合【免疫】 |
| 依赖增加冷却的打法 | 【免疫】状态下不会被增加冷却 |
| 低速队伍 | 容易无法及时消耗【缚魂】层数，导致到期引爆 |

## 配装建议

### 输出位配装方向

优先考虑：

- 【虚】相性直伤输出
- 高速或能自拉条的输出
- 不依赖持续伤害
- 不依赖控制命中
- 能在开场免疫期结束前后保持稳定输出
- 能承受点名或随机单体高倍率伤害

推荐属性倾向：

- 输出主属性按角色自身倍率决定
- 暴击率 / 暴击伤害适合直伤角色
- 速度可提高出手频率，有助于消耗【缚魂】层数
- 生存词条也有价值，避免被点名单体或群伤压死
- 不建议为了持续伤害效果投入资源

### 辅助位配装方向

优先考虑：

- 速度
- 生存
- 效果抵抗
- 净化频率
- 免疫覆盖率
- 拉条能力
- 治疗量 / 护盾量

辅助位主要任务：

1. 提前覆盖免疫，减少【石化】和【速度下降】影响。
2. 及时净化被【石化】或【速度下降】影响的核心角色。
3. 通过拉条或加速提高我方行动频率，帮助消耗【缚魂】。
4. 维持队伍在高倍率防御伤害和真实伤害威胁下的存活。

### 生存位配装方向

优先考虑：

- 生命
- 防御
- 减伤
- 护盾
- 治疗
- 效果抵抗
- 速度

注意事项：

- Boss 技能伤害多次以防御倍率计算，生存压力较高。
- 【石化】会导致角色无法行动，且回合结束时无法自动恢复冷却，队伍不能只靠慢速治疗硬扛。
- 【缚魂】到期会造成全体真实伤害，需要通过行动频率消耗层数，而不是单纯堆防御。

## AI 调号器判断规则

当用户询问“这个 Boss 怎么打”“这个角色能不能打柯洛罗修罗”“我的配装怎么调”时，AI 应按以下顺序判断：

1. 判断角色是否属于【理】相性：若是，需要提醒会受到额外200%伤害，生存压力极高。
2. 判断队伍是否有【虚】相性直伤输出：若有，应优先作为核心输出候选，因为修罗难度下对柯洛罗伤害增加100%。
3. 判断队伍是否依赖控制：若依赖控制，需要提醒 Boss 和恐鸦免疫所有控制效果。
4. 判断队伍是否依赖减少行动进度：若依赖推条、偷条、击退行动进度，需要提醒 Boss 和恐鸦免疫减少行动进度效果。
5. 判断队伍是否依赖持续伤害：若依赖 Dot 或【持续伤害】，需要提醒 Boss 和恐鸦免疫持续伤害。
6. 判断队伍是否依赖开局减益：若依赖开局破防、增冷却或其他减益，需要提醒柯洛罗开场有3回合【免疫】。
7. 判断队伍是否能处理【石化】：若没有免疫、净化或较高效果抵抗，容易被控断循环。
8. 判断队伍行动频率是否足够：若队伍过慢，可能无法及时消耗【缚魂】层数，导致全体真实伤害爆发。
9. 判断队伍是否有足够生存：Boss 有群体多段防御伤害、随机单体600%防御伤害和缚魂真实伤害。
