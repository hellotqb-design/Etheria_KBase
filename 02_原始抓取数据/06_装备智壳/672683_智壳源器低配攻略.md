# 智壳源器低配攻略

**FBI WARNING**

本攻略主要展示不用SSR辅助的前提下通关各智壳、源器副本的可能性。

最初做这个攻略的时候梨梨还没有白送，现在正常是不用这么辛苦的玩法了，不过新玩家看看还是能有点启发作用。

魔↗术↘技↘巧

由于常规情况下我们的主要伤害来自于C位的大招，因此C位放大招的节奏在很大程度上决定了通关的时长，从而进一步影响配队的攻防倾向，这里我们设定一个概念【单C通关基准大招数】，也就是满血boss能吃你的主C大招几下，这个数值只要看首回合伤害粗略估计一下即可。

这个数值的作用是快速推算当前队伍通关需要的回合数，因为放大招的节奏是基本固定的，这里我们以555莉拉姆、551梨梨为例，来对比一下有没有梨梨的前5回合行动区别。

有梨

回合1：梨梨技能2 >>> 莉拉姆拉条30% >>> 莉拉姆大招

回合2：梨梨技能3 >>> 莉拉姆拉条30%+大招冷却 >>> 莉拉姆大招

回合3：梨梨技能1 >>> 莉拉姆拉条30%（还有几率额外拉条） >>> 莉拉姆技能1助战 >>> 莉拉姆技能2

回合4：梨梨技能2 >>> 莉拉姆拉条100% >>>莉拉姆技能1

回合5：梨梨技能1 >>> 莉拉姆拉条30%（还有几率额外拉条） >>> 莉拉姆技能1助战 >>> 莉拉姆大招

总计莉拉姆放了3个大招、3个技能1、1个技能2

无梨

回合1：莉拉姆大招

回合2：莉拉姆技能2

回合3：莉拉姆技能1

回合4：莉拉姆大招

回合5：莉拉姆技能2

总计莉拉姆放了2个大招、1个技能1、2个技能2

注意并不是所有C大招都是3回合冷却（虽然现在最常用的几个都是），更长的冷却时间会导致梨梨的技能3不足以让主C大招在回合2再次可用，从而改变上述行动顺序，除非梨梨技能3升满，请自行注意实际使用的主C大招冷却回合数，本攻略默认不涉及此种例外情况。

比如你的主C大招能砍boss满血的60%，则需要放两次才能通关，有梨的情况下需要2回合，无梨的情况下需要4回合。

比如你的主C大招能砍boss满血的40%，则需要放三次才能通关，有梨的情况下需要5回合，无梨的情况下需要7回合。

推算出回合数之后，首先可以判断能不能通关，因为有的本不允许活很久，其次可以判断能不能组不超过2回合的速通队，最后可以判断阵容的调整余地。

本攻略以1~2回合为速通队，4~-5回合为短线队，7~8回合为长线队。日常8本全部可以速通，大部分能用短线队，少部分能用长线队。

其中速通队是一种比较特殊的情况，只要不被秒就能过，因此几乎不用考虑生存问题，而是更注重爆发性。长线队练度提高后能变成短线队，但短线队提高练度未必能变成速通队。

低配玩法的重点在于设法减少回合数，把长线队转换成短线队，短线队转换为速通队，从而化不能打为能打，化翻车为不翻车。而要想把同样的大招数压缩进更少的回合内，一个思路是带双C，另一个思路是减CD，减CD可以通过技能（如梨梨、彼安汀、米娅、水生绫、月夜见、寻愿多乐丝、R卡奇、柯洛罗、风见一宵）、智壳（如万魔、万圣节）、矩阵（如时光驾驭）等多种方式达成。

尼可娜娜（炼狱）

对练度要求最低的炼狱本，如果事先准备好相关角色，50级开放时可以马上通关，保证日常体力不浪费。

boss初始处于减伤状态，我方每高1点速度破1.5%减伤，最多有效40点。不能打速通的话最好有减速手对冲小怪给boss的加速，速通有加速手就够了，正常能用的角色太多了，初始必刷的逆翎+梨梨直接就是一对。

低配能通关的阵容仍然很多，这里只列出部分角色：

加速：风见一宵、妮璐

加速C：莉拉姆、砺行阿凛

减速：风见一宵、蝰蛇、方格斯

生存：恶命、缬草

替补：戍守卡奇、黛纳

多琪（炼狱）

敌方初始处于减伤状态，吃到控制类debuff会移除三回合，但已经被移除时再次吃控制技不会延长。首先保证每面最多两个大招解决，生存就基本没问题，然后稍微注意一下控制位的对轴，不要空放大招就行。

核心对策是控制手，首选酣梦一宵，一是有睡眠增伤，二是大招CD比较短。

理想情况下酣梦和主c分别是第四、五动，让主C吃到睡眠增伤，大招秒掉第一面，酣梦带万魔，第二面再重复一次，也就是速通队的打法。

重构兵祸（炼狱）

重点机制是敌方开局有一个3回合的免疫，常规打法（包括DOT队）刚需一个驱散手，正常玩法还是直接用逆翎就行，没逆翎的也有一堆驱散和偷窃能用，除了水系有啥用啥。

低配虽然看上去也有好几个驱散手，但真正好用的只有凛心黛纳。

没驱散手的邪修思路是绕过免疫，只带寻愿多乐丝、幻音千明、暗卡奇等buffer，上位buffer还有梨梨、多琪、水生绫、拂秋、彼安汀等。

严格来说暗卡奇是混合型辅助，但能延长队友的buff，在组buffer队时可视为特殊的专拐。

低配通关效果如下所示，图一是纯SR的常规驱散打法，风见可换妮璐，如果换梨梨可以直接二回合秒了。图二是无驱散双单C直击boss本体，图三是先清攻击之手的安全打法，带群C是为了同时给本体扣血。

作为源器的入门本，兵祸的通关阵容非常灵活，只要有驱散手，既可以单C+强增伤，又可以双C+弱增伤，也可以全DOT，还可以直伤+DOT混合，甚至可以嘲讽住慢慢磨死，正常游玩从逆翎出生后基本就是无脑过了，低配到SS4.1凛心黛纳出生后也同样无脑过了。

莎朗（炼狱）

boss本体不吃debuff，大量毒伤，解法是速通、免疫、强力净化三选一。

速通无视机制，输出够高就行。

免疫只有五个角色，三个还是被克的水系（但宁芙涅速通能用），有千羽随便打，暗卡奇只能尝试打短线。

强力净化如桑提克、多琪等可以拖长战线增加容错率，但角色本身养成偏后，真能上场时不出意外已经能速通了。

赫妍（炼狱）

boss本体约等于白给，真正的威胁来自于小怪，所以传统的重点机制是控小怪，亲妈对策是席琳斯，现代打法是控一下破减伤，然后直接秒。

另外这是DOT队能打的三个本中最有价值的，一是难度最高，二是产出顶级，三是机制有利，DOT不被分摊伤害和反击，队员还能堆点血防，在近代阶段也是本图的主流解法，直到逆翎、莉拉姆这一代人权出现，统治地位才被直伤队夺走。

低配的主要痛点是控制手，R卡奇、奥波尔、克拉肯、绯月鞠都无法稳定控住小怪，顶多带万魔多控一回合，战线稍长点主C就容易被反击死，基本只能打速通，最快首回合2大招直接带走，图一是多打一次尾刀的情况，此时R卡奇起了拉满条的额外作用，但换成其他控制手也一样过。

顺便一提，有个流言是错误的，本关不是没有暗善雅就不能用DOT队，奥波尔+方格斯+梨梨+凌罗就有九成以上胜率，甚至不需要任何三矩阵，不需要胡狼，不需要万魔，效果见图二，测试用队伍一速275，技能是凌罗111，梨梨551，两个SR555。

奥洛拉（炼狱）

非高难本的毕业考核关，数值和机制难度都比之前有提升，甚至从某种意义上来说不次于之后的高难关，刷迅光的最大障碍。

核心机制是boss会释放一个相当于我方50%最大生命的光环，需要施加等量的治疗才能破除，否则会减伤90%，而且破除用的部分不会加血。

但我自己计算正好是60%最大生命

注意奶妈一般带超新星加奶量，而超新星没有双速矩阵，实战中很容易造成梨梨把主C拉到奶妈之前的局面，所以本关要尽量抬奶妈速度，实在抬不到主C之前再压主C速度。如果梨梨点了二技能，这个差距还要制造得更大才行。

这关尽量打速通，虽然能打长线但不像之前的本一样可以预计要打多长，因为光环冷却只有两回合，再加上周期性免疫，一旦脱离速通窗口，再想让光环、奶妈大招、智壳技能、C大招、驱散、破甲全对上轴难如登天。

Boss除了直伤之外，还有一个狂言DOT会造成每回合掉血，而且一挂就是全体三回合，需要额外配奶妈或净化，三回合免疫也是会反复挂的，这些同样是不利于长线作战的因素。

作为核心对策，首先要找个能大招一次破掉光环的奶妈，凌罗、米娅、千明、希尔芙等非木属性奶妈的大招基础奶量都是20%~30%，有的还有额外治疗机制，比如凌罗是基于buff数，千明是基于已损失血量，希尔芙没有，米娅也没有并且奶量基于自身血量计算。

奶量计算公式据我实测是基数*次数*基础比例*(1+技能描述加成)*(1+智壳加成+源器加成)

以超新星+8治愈的凌罗用没升级的三技能治疗3buff的主C为例，奶量=主C总血量*2*12%*(1+15%*3)*(1+35%+25%)=主C总血量*55.68%

如果按照boss技能描述的话本来应该是够破50%光环的，但破60%就不够了，并且凌罗如果不点3技能和超新星对轴的话，长线没吃到超新星加成的时候只有43.5%

在55.68%的基础上加进化是59.856%，没用。

在55.68%的基础上加智壳第二被动，13%~15%可以高于60%

在55.68%的基础上升三技能，一点就能高于60%，升到四级是71.68%

好在凌罗还会挂一个鲸祈buff，在主C行动之前会再治疗一次至少5%，加上55.68%正好够。

满三技能希尔芙同装备下奶量是固定48%，就算加15%智壳被动和进化，最多也只有56.1%，没用。

满三技能千明同装备下基础奶量也是48%，需要主C血量约为58%以下时才能破掉，速通在首回合可以实现，非速通后面无法保证。

米娅要分情况讨论，速通只要奶妈功能，此时自身血量基数很大，不用点技能，并且智壳和矩阵中只补一种基本就够了，另一种可以补速度，非速通要堆命中，不点技能奶量低于吃满buff加成的凌罗，并且同样有和超新星错轴的问题，好处是自带驱散。

总而言之，速通除了希尔芙都能带，非速通又不投资没有绝对稳的，此时凌罗点满三技能会舒服很多，但等能速通又变多余了，自己决定吧。

正常速通有单C（包括谛听）就够，主要考察辅助是否齐全，尤其是有没有逆翎、克索司这样能同时驱散+破甲的角色，有的话能省出一个增伤辅助位置，速通就比较容易达成。反之则主要考察双C强度，辅助可以不需要SSR，但双C爆发力必须顶级，基本上要莉拉姆、夜离、贝茨比、谛听四选二，而且贝茨比现在已经不太用了。如果没刷初始导致两边都不达标的话，这里再额外附赠两个牢队阵容，但翻车率可能会有一半。

【逃课技巧】在必须速杀但只差一点血量杀不掉的场合，可以把主暴率的源器换成主暴伤凹暴击。

炼狱津（炼狱）

日常8本里的首个高难本，boss会改变属性并在我方同属性角色行动四次后释放额外攻击，因此我方属性要尽量分散避免暴毙，除非能打速通，而且boss也比之前的肉很多，打速通也更难。低配主要是靠蝰蛇+木善雅苟住，其余两个位置增伤+输出即可。

![image](https://cdnimg-v2.gamekee.com/wiki2.0/images/w_2780/h_1264/50364/154257/2025/11/1/561467.jpg)

没啥难度，机制拆解如下：

二技能：减速+自身5层【巨型帷幕】（首段攻击90%减伤）

一技能：破甲

被动：推条+按【巨型帷幕】层数增伤

小怪1：boss一层【巨型帷幕】+拉条

小怪2：boss一层【巨型帷幕】+破甲

整体思路是多动消【巨型帷幕】层数，带个减速手减缓boss叠层，带净化、加速、加防、减攻保障生存，破甲一般都有。

智壳攻击也消层数并且抵减防，使得角色攻击不受【巨型帷幕】影响（但偶尔有bug不生效），所以最差的结果无非是平时打不动全靠开大，正常有梨梨开大也比较快，只要苟住不死就行。

正常打法下完全不不需要配稀奇古怪的队，逆翎+梨梨+凌罗的标准阵容已经轻松满足这点要求，带个荼蘼、玄骊这种多动C无脑乱杀。面板

够高的莉拉姆更是直接一刀砍穿，不知机制为何物。

视频里使用了尽可能低配的阵容，无SSR，无觉醒SR，除谛听外无光暗SR，原则上用砺行阿凛应该也行，因为这是测试用的小号，没练砺行阿凛，装备也是有啥用啥没刻意调，风见一宵都没双速，刻影也没怎么用，正常玩家用这个阵容打的话还会更快更稳一些。
