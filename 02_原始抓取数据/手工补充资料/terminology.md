## 玩家黑话 / 常见简称

下面这部分主要收社区常见说法，默认不等于官方正式用词，但在检索和问答里非常常见。

### 养成和账号类

| 说法 | 含义 | 备注 |
|---|---|---|
| 开荒 | 新手期、推主线和初步刷本阶段 | 常和“前期 / 中期 / 后期”一起出现 |
| 长草 | 后期资源节奏放缓、日常挂机为主的阶段 | 也常写成“养老期” |
| box | 账号持有角色池、整体卡池宽度 | 常见说法如“box 不够宽” |
| 练度 | 角色或账号的养成完成度 | 包含等级、技能、源器、智壳、词条 |
| 毕业 | 达到阶段性理想配置 | 可能指角色毕业、装备毕业、智壳毕业 |
| 胚子 | 有潜力继续强化的装备或智壳 | 常见说法如“刷到好胚子” |
| 本体可用 | 不依赖高原质或高技能等级，也能先上场 | 常用于评价辅助和功能卡 |
| 满技 | 关键技能点满 | 常见如“3 技能满”“终结技满” |

### 战斗和配队类

| 说法 | 含义 | 备注 |
|---|---|---|
| 主 C | 队伍核心输出 | “C” 即 carry |
| 副 C | 补伤害或副核输出 | 常和主 C 搭配 |
| 拐 | 给输出增益的辅助位 | 如加攻、暴伤、拉条、减冷却 |
| 生存位 | 治疗、护盾、免疫、抗压角色 | 不一定纯奶 |
| 抗压 | 扛住敌方伤害和机制压力 | 常见说法如“先上治疗抗压” |
| 补位 | 根据副本需要补一个功能位 | 例如破防、控制、驱散 |
| 直伤 | 依赖常规倍率和面板的直接输出 | 对应 Dot |
| Dot | 持续伤害体系 | 常被当作逃课思路 |
| 逃课 | 用特殊机制绕开常规数值压力 | 如无限推条、挑衅、真伤 |
| 拉条 | 提高我方行动进度 | 也常写成“推我方条” |
| 推条 | 一般指推动行动条 | 语境里可能是拉我方，也可能是推敌方 |
| 偷条 | 压低敌方行动进度，变相把回合“偷”过来 | 常见于减速 / 推条角色 |
| 控条 | 围绕行动条做控制 | 包含拉条、偷条、减速 |
| 配速 | 调整队伍速度顺序 | 常见句式“先配速再谈伤害” |
| 对轴 | 把出手顺序、技能循环、智壳触发对齐 | 尤其常见于高难本和活动 |
| 双暴 | 暴击率 + 暴击伤害 | 输出配装高频词 |
| 破防 / 破甲 | 施加防御下降类效果 | 社区里常混用 |
| 先机 | 开局立即行动或开局抢节奏的机制 | 常用于评价某些治疗 / 辅助 |

### 副本和玩法类

| 说法 | 含义 | 备注 |
|---|---|---|
| 源器本 | 源器副本 | 常指兵祸、多琪、奥洛拉等 |
| 智壳本 | 智壳副本 | 常指凶影追缉系列 |
| 月塔 | 虚烬探索 | 当前是稳定可确认的别名 |
| 作业 | 别人的现成阵容和配置参考 | 常见“抄作业” |
| 抄作业 | 照搬现成阵容和配装 | 对新玩家很常见 |
| 凹分 | 反复刷更高分数 | 常见于活动榜单、后期玩法 |
| 低保 | 保底拿到目标排名或奖励 | 常见于 RTA / 榜单玩法 |
| 人权卡 | 泛用性极高、缺了会明显难受的角色 | 社区评价，不是官方标签 |
| 国家队 | 某环境下最稳定、最常见的一组强势组合 | 版本性很强 |

### 问答时的处理原则

- 如果用户用黑话提问，优先先翻译成标准描述再回答
- 如果黑话可能有歧义，要说明它在当前语境里的含义
- `box / 人权卡 / 国家队 / 毕业` 都属于强社区语境词，不要当作官方口径
