## 一句话总结

刺血大公·赫尔基德在炼狱难度下是一个高压群体输出型 Boss。它和役血灵免疫控制、免疫减少行动进度、免疫持续伤害，并且会通过【巨型帷幕】获得首段伤害减免与伤害提升。应对思路是：不要依赖控制、推条或 Dot，优先使用直伤、多段技能或高频技能输出，并准备净化、免疫、治疗、护盾或减伤来应对群体伤害和减益压力。

## 炼狱难度特殊规则

### 原始规则文本

炼狱难度下，赫尔基德和役血灵免疫所有控制效果、减少行动进度效果和【持续伤害】效果，并对【虚】相性异格者造成额外200%伤害。

赫尔基德将有概率产出拥有3条矩阵的智壳。

赫尔基德与铁幕之中沉睡的力量融为一体，化身为完全形态——刺血大公；睥睨万物的至高意志，将为世界降下毁灭。

### 机制拆解

| 机制 | 说明 |
|---|---|
| 免疫控制 | 赫尔基德和役血灵免疫所有控制效果 |
| 免疫推条 | 赫尔基德和役血灵免疫减少行动进度效果 |
| 免疫持续伤害 | 赫尔基德和役血灵免疫【持续伤害】效果 |
| 虚相性惩罚 | 赫尔基德和役血灵对【虚】相性异格者造成额外200%伤害 |
| 智壳掉落 | 有概率产出拥有3条矩阵的智壳 |

## 技能信息

## 技能一：恶堕黑赫

### 原始技能信息

- 技能名称：恶堕黑赫
- 技能标签：群伤 / 减益
- 技能效果：对敌方全体造成500%伤害，100%概率为目标附加2回合【防御下降】。

### 关联状态：防御下降

- 状态名称：防御下降
- 状态效果：防御下降60%。

### 机制解读

该技能是群体伤害加防御削弱。由于【防御下降】会让后续群伤更危险，因此队伍需要具备净化、免疫、护盾、减伤或高生存能力。

---

## 技能二：巨型帷幕

### 原始技能信息

- 技能名称：巨型帷幕
- 技能标签：群伤 / 减益
- 技能效果：对敌方全体造成600%伤害，100%概率为目标附加2回合【速度下降】，并为自身附加5层的【巨型帷幕】。

### 关联状态：速度下降

- 状态名称：速度下降
- 状态效果：速度降低30%。

### 关联状态：巨型帷幕

- 状态名称：巨型帷幕
- 状态效果：持续期间，攻击自身的首段伤害减免90%。最多附加10层，不可被驱散，每受到1次技能伤害减少1层。

### 机制解读

【巨型帷幕】是本 Boss 的核心机制。它会大幅降低攻击首段伤害，并且不可驱散，只能通过技能伤害逐层消耗。单段爆发会被明显克制，多段技能或高频技能伤害更适合处理该机制。

---

## 技能三：荆棘

### 原始技能信息

- 技能名称：荆棘
- 技能标签：被动
- 技能效果：赫尔基德攻击时有50%概率击退目标15%行动进度，并且自身每有1层【巨型帷幕】，伤害提升10%。

### 机制解读

该被动使 Boss 在攻击时具备行动条干扰能力，同时【巨型帷幕】层数越高，Boss 输出越高。若放任【巨型帷幕】长期存在，Boss 会同时获得减伤和增伤，队伍压力会快速上升。

## 应对思路

### 推荐队伍方向

| 推荐方向 | 原因 |
|---|---|
| 高生存队伍 | Boss 群体伤害倍率高，还会附加【防御下降】 |
| 净化 / 免疫角色 | 用于处理【防御下降】和【速度下降】 |
| 护盾 / 减伤 / 治疗 | 用于承受连续群体伤害 |
| 多段技能输出 | 可以更快消耗【巨型帷幕】层数 |
| 高频技能输出 | 【巨型帷幕】每受到1次技能伤害减少1层 |
| 非 Dot 直伤输出 | Boss 免疫【持续伤害】 |
| 不依赖控制的输出体系 | Boss 免疫所有控制效果 |
| 不依赖推条的队伍 | Boss 免疫减少行动进度效果 |

## 配装建议

### 输出位配装方向

优先考虑：

- 多段技能输出
- 高频技能输出
- 直伤输出
- 不依赖持续伤害
- 不依赖控制命中
- 能在【巨型帷幕】存在时稳定消耗层数

推荐属性倾向：

- 输出主属性按角色自身倍率决定
- 暴击率 / 暴击伤害适合直伤角色
- 速度可提高技能循环
- 不建议为了持续伤害效果投入资源

### 辅助位配装方向

优先考虑：

- 速度
- 生存
- 效果抵抗
- 治疗量
- 护盾量
- 净化频率
- 免疫覆盖率

辅助位主要任务：

1. 保证队伍不被【防御下降】后连续击杀。
2. 处理【速度下降】导致的行动节奏崩坏。
3. 帮助输出位尽快消耗【巨型帷幕】。
4. 维持队伍在高压群伤下的存活。

### 生存位配装方向

优先考虑：

- 生命
- 防御
- 减伤
- 护盾
- 治疗
- 效果抵抗

注意事项：

- Boss 会附加【防御下降】，纯堆防御可能不稳定。
- 队伍最好具备净化或免疫，否则防御下降后容易被连续群伤击穿。
- 如果角色依赖速度循环，需要注意【速度下降】会降低行动频率。

## AI 调号器判断规则

当用户询问“这个 Boss 怎么打”“这个角色能不能打赫尔基德炼狱”“我的配装怎么调”时，AI 应按以下顺序判断：

1. 判断角色是否属于【虚】相性：若是，需要提醒会受到额外200%伤害，生存压力极高。
2. 判断队伍是否依赖控制：若依赖控制，需要提醒 Boss 免疫所有控制效果。
3. 判断队伍是否依赖减少行动进度：若依赖推条、偷条、击退行动进度，需要提醒 Boss 免疫减少行动进度效果。
4. 判断队伍是否依赖持续伤害：若依赖 Dot 或【持续伤害】，需要提醒 Boss 免疫持续伤害。
5. 判断队伍是否有多段技能或高频技能伤害：若缺少多段技能，处理【巨型帷幕】会较慢。
6. 判断队伍是否有净化或免疫：若没有，容易被【防御下降】和【速度下降】拖垮。
7. 判断队伍生存是否足够：Boss 有500%与600%群体伤害技能，且【巨型帷幕】层数会提高 Boss 伤害。
