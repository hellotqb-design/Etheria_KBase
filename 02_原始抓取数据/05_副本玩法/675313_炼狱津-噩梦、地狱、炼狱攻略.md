# 炼狱津-噩梦、地狱、炼狱攻略

![image](https://cdnimg-v2.gamekee.com/wiki2.0/video/50364/889292/2025/7/27/189670.mp4)

重点：别上同属性的角色。Boss伤害很高。要么磨，要么速通。

噩梦：

对于噩梦难度来说，没有难度，不需要考虑机制，用常见的破甲或者辅助+主C+梨梨+随便一个奶妈就能过。或者说有点难度的队伍都可以无视机制过关。切记主C在梨梨或者别的辅助后面出手，破甲手在主C出手前破甲就行了。

![image](https://cdnimg-v2.gamekee.com/wiki2.0/images/w_437/h_114/50364/2025/8/27/efc8ac51-cff9-481f-83fd-85e10d677ae6png)

地狱：

从地狱开始就需要考虑副本的机制了。

BOSS的核心技能：

![image](https://cdnimg-v2.gamekee.com/wiki2.0/images/w_580/h_264/50364/2025/8/27/9328609c-9054-4bcc-acdb-79f0dd8ac9f3png)

同属性的我方角色累计行动就会被点名，造成伤害并且附上不可驱散的debuff。

![image](https://cdnimg-v2.gamekee.com/wiki2.0/images/w_310/h_334/50364/2025/8/27/0dc9b290-6c89-4c93-9ca4-9b3529d6863apng)

假如在有debuff的情况下被“欲火烈旋”击中，而且自身刚好是被克制的相性，则有很大概率暴毙。

所以我们需要了解各种目前能使用的角色：

| C | 奶 | 辅助 |
| 恒定 | 夜离 | 恶命 |
| 破序 | 莉拉姆、谛听 | 梨梨 |
| 理 |
| 虚 | 善雅 | 克洛斯 |
| 异 | 凌罗 | 蝰蛇 |

因为正常不能属性相同，所以可以选择的角色配队其实很少。速刷队因为是可以秒boss所以是论外，最后单独讨论。

正常配队即为：

A队：夜离、梨梨、善雅、蝰蛇

B队：夜离、梨梨、克洛斯、凌罗

C队：莉拉姆（谛听）、恶命、善雅、蝰蛇

D队：莉拉姆（谛听）、恶命、克洛斯、凌罗

到这一步只要配队逻辑正常，一般就能挂机过了。

炼狱：

到了炼狱，你需要注意这一步主要考验的是我方角色的整体练度。需要三条智壳的就去刷能刷的三条智壳。

智壳选择：

| 主C | 启示录（多次伤害主C首选：夜离）、白獠武者（单体单发：谛听、莉拉姆）、伯爵（提升攻击+吸血） |
| 辅助 | 琥珀（免死一次）、墨月（延长DEBUFF）、万圣节（刷新技能）。天角兽（驱散沉默）、灯塔（受击治疗拉条）、胡狼卫士（拉条+速度BUFF）、匠（防御力转化为巨额护盾） |
| 奶妈 | 超新星（爆发治疗）、最强音（治疗储存） |

套装12件套+8件套+6件套，所有角色能穿就尽量穿最好的，智壳的被动效果也要尽量洗到最好。炼狱炼狱津是对整体PVE的最终考试，马虎不得。
